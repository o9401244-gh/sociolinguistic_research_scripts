# Sociolinguistic Research: Facilitating Analysis with Artificial Intelligence
## Python Script Repository

**Authors:** Ray G Opao, Mennen Pearl C Talibong, Marivel V Burgos

This repository contains all Python scripts referenced in the book
*Sociolinguistic Research: Facilitating Analysis with Artificial
Intelligence*. Scripts are organised by chapter and listing number,
mirroring the book's structure.

A citable archived version of this repository is available at:
**https://doi.org/10.5281/zenodo.PENDING**

---

## Requirements

- Python 3.12 or higher
- pandas
- numpy
- matplotlib
- scipy
- statsmodels
- scikit-learn

Install all dependencies with:

    pip install pandas numpy matplotlib scipy statsmodels scikit-learn

Not all packages are needed for every method. If you prefer to install
only what you need, run the relevant script first and install any
package that raises an ImportError.

## Usage

Each script is documented in the corresponding chapter of the book.
Scripts are designed to be used in sequence:

1. Run `listing_3_1_prerequisite_preprocessing.py` first --
   select your cluster and method from the interactive menu
2. Run the cluster-specific processing script (e.g., `listing_4_1_...`)
3. Run the cluster-specific analysis script (e.g., `listing_4_2_...`)

Output files follow the naming convention:

- `out_<cluster>_<slug>_preprocessed.csv` -- after Step 1
- `out_<cluster>_<slug>_processed.csv` -- after Step 2
- `out_<cluster>_<slug>_analyzed.csv` -- after Step 3

**Note:** Three methods use their own raw input files instead of
passing through Step 1: Language Prestige Analysis (Listing 10.7),
Language Trend Forecasting (Listing 10.9), and Computational
Simulation of Language Change (Listing 10.3). See the book and
script comments for details.
